# AI Resume Screening API

This project is a FastAPI-based Resume Screening System that analyzes resumes and compares them with job descriptions.

## 🚀 Features
- Upload resume (PDF)
- Compare with job description
- Generate match score
- Extract relevant keywords

## 🧠 Tech Stack
- Python
- FastAPI
- Scikit-learn
- NLP (TF-IDF, Cosine Similarity)

## ▶️ How to Run
```bash
uvicorn main:app --reload
